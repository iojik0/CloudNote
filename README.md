## 🚀 Features

- ✅ User registration & login (with password hashing)
- 📅 Calendar view for notes
- 🔗 Note sharing between users
- 💾 Database persistence
---
## 🛠️ Tech stack

- **Language:** Java 17+
- **UI framework:** JavaFX
- **Build tool:** Maven
- **Database:** MySQL
---
## 📦 Getting started

### Prerequisites
- JDK 17 or later
- Maven

### Installation & run

    ```bash
    # Clone the repository
    git clone https://github.com/iojik0/CloudNote.git
    cd CloudNote

    # Build the project
    mvn clean compile

    # Run the app
    mvn javafx:run
> 💡   Replace mvn javafx:run with your actual run command if different (e.g., mvn exec:java).
