# CloudNote

**CloudNote** — это ваше личное приложение для заметок с интеграцией облачных технологий.

## 🚀 Возможности

- Создание и редактирование заметок
- Синхронизация с облаком
- Поддержка Markdown
- Минималистичный и стильный дизайн в тёмной теме

## 🛠️ Установка и запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/ваш-логин/CloudNote.git
2. **Откройте проект в IntelliJ IDEA**
3. **Запустите приложение:** Найдите и выполните главный класс приложения (например, Main.java)
